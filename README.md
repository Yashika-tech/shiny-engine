# shiny-engine
How to use it:

Open the Copilot Chat panel/Claude panel and click the Attach prompt button (or type /update-cv-for-job)
Replace [PASTE JOB DESCRIPTION HERE] with the job posting
Replace [PASTE YOUR CV HERE] with your current CV text
The prompt will produce:

A fully rewritten, ATS-optimized CV aligned to the role
A summary of every change made
A gap analysis highlighting missing requirements and how to address them
